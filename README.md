## getting started

The easiest way to manage node versions is with a version manager such as `nvm`.  
We leave it to the developer to install `nvm` or implement another method which can access the `.nvmrc`.  

```code
cd conex && 
nvm use &&
yarn && 
yarn start:debug &&
chrome localhost:1337
```
